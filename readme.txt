Link an entire text widget to a specific URL.

== Description ==

This plugin will create a text widget which will be wrapped in a link to a URL of your chosing.  


== Installation ==

1. Upload Text Widget As Link to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the widget to a widget area an provide the text content and the url it should link to
